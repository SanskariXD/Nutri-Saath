# Nutri Saath

Nutri Saath is a web app that helps families scan packaged foods, understand labels, and make safer choices. This repository contains the Vite + React frontend and a Node/Express backend in `label-backend/`.

## Tech Stack

- Vite, React, TypeScript
- Tailwind CSS, shadcn-ui
- Node.js, Express, MongoDB (optional)

## Local Development

Frontend:

```bash
npm install
npm run dev
```

Backend:

```bash
cd label-backend
npm install
npm run dev
```

## Build

```bash
npm run build
```
